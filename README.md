# dotfiles_220

Basic dotfiles from some x220 machine.

This repo contains config files for

* bash

* i3

* mpdnotify

* oh-my-zsh (custom files)

* synaptic config

* Xresources, Xdefaults

* gtk-2.0 & 3.0

* livestreamer

* mps-youtube

* ncmpcpp

* ranger

* tmux

* zsh

* compton

* mpd

* mpv

* neofetch

* sublime3 packages

* xfce4 configs
